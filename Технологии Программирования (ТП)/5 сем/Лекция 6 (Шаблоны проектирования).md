#Соло_утомила
* Есть метода "Жизненный цикл программной продукции". Игра будет по ней
* Вопросы будут строиться только по разделам, упомянутым ниже:
	* Жизненный цикл как основа системного подхода к разработке программной продукции
		* Жизненный цикл состоит из фаз, которые делятся на стадии, которые делятся на этапы:
			* Фаза исследования (состоит из стадий по разведке рынка и требований заказчика)
			* Фаза проектирования(состоит из стадий по разработке ТЗ и  проектирования)
			* Фаза разработки (состоит из стадий по написанию кода, его тестирования итд)
			* Фаза эксплуатации (состоит из стадий эксплуатации и сопровождения)
	
	* Техническая документация программной продукции
		* К ней относятся: стандарты, рук. документы итд
	
	* Стандарты (ГОСТ, IEEE итд)
